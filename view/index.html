Create a New User

POST /api/exercise/new-user

Add Exercises

POST /api/exercise/add

exercise-tracker restFUL API

  • A user can be created by POSTing to /api/exercise/new-user. An object will be returned with 2 properties: username & _id.
  • A list of all users can be retrieved by sending a GET request to /api/exercise/users
  • You can add an exercise to any users record by POSTing form data (userId, description, duration (mins), date (yy-mm-dd) to /api/exercise/add. If no date is given, current date will be used. A user object will be returned with the exercise fields added.
  • A full exercise log of any user can be retrieved by sending a GET request to /api/exercise/log with a parameter of userId(_id). An object will be returned with the exercise log for that user.
  • You can retrieve a partial log any user by also passing along optional parameters of from, to or limit.
